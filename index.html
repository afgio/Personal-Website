Jeopardy! Quiz

Jeopardy! Quiz

This app uses Axios to make API calls to the jservice API. When users click the "New Question" button, a GET request is made via an Axios call to the API which returns a JSON object containing a question and other relevant data such as the correct answer as well as the question's point value. This data is then displayed to the player to simulate a Jeopardy! quiz. Test your skills and set a new high score!

Alexandria

Alexandria

Alexandria is an onchain library for lending and borrowing content. Users borrow NFTs which permit access to books and journals. Alexandria contracts allow anyone to borrow books for however long they like (assuming sufficient funds) whilst rewarding authors royalties from payments. We help authors ensure they get paid each time their books are read.
